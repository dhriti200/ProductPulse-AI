# 🎵 ProductPulse AI

> An AI-powered Product Analytics & User Feedback Intelligence Platform built using Streamlit, Python, Plotly, and Hugging Face Transformers.

## 📌 Overview

ProductPulse AI helps Product Managers and Product Analysts understand user behavior and customer feedback through interactive dashboards, AI-powered review analysis, and product prioritization.

The project combines:
- Product Analytics
- Customer Feedback Analysis
- AI-based Review Understanding
- Feature Prioritization

using real Spotify review data and a synthetic product usage dataset.

# ⚙ Installation

Clone the repository

```bash
git clone https://github.com/yourusername/ProductPulse-AI.git
```

Move into the project

```bash
cd ProductPulse-AI
```

Install dependencies

```bash
pip install -r requirements.txt
```

Run the application

```bash
streamlit run app.py
```
